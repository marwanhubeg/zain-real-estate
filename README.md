# منصة عقار زين المتكاملة\n\nهذا المستودع يمثل منصة عقار زين المتكاملة، والتي تجمع بين الواجهة الأمامية والخلفية لتوفير حلول عقارية ذكية مدعومة بالذكاء الاصطناعي.\n\n## المكونات الرئيسية\n\n*   **الواجهة الأمامية (Frontend):** لتفاعل المستخدم وعرض العقارات.\n*   **الواجهة الخلفية (Backend):** لإدارة البيانات والمنطق البرمجي.\n*   **الذكاء الاصطناعي (AI):** لتحليل البيانات وتقديم توصيات ذكية.\n\n## كيفية التشغيل\n\nيرجى الرجوع إلى الوثائق الخاصة بكل مكون (الواجهة الأمامية والخلفية) للحصول على تعليمات التشغيل التفصيلية.\n
